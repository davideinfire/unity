# Daily Journal

A small personal journaling application built with Python.

The purpose of this project is to organize daily notes, record moods, and browse previous journal entries.

---

## Sample Session

```
July 15

Mood: 😊 Happy

Today I finished my Python project
and went for an evening walk.

----------------------

Entries stored: 3
Average mood: Positive
```

---

## Project Files

entry.py
Represents a journal entry.

journal.py
Stores all journal records.

timeline.py
Sorts entries by date.

mood.py
Calculates mood statistics.

writer.py
Prints journal pages.

archive.py
Displays saved entries.

demo_entries.py
Example journal records.

---

## Run

python app.py

---

No external libraries required.
